{% comment %} Reusable visitor-map block (clustrmaps). Renders the placeholder div + a deferred loader script. Pull into any page via the Liquid include tag pointing at this filename. Currently included from `_includes/body/footer.html` (our project shadow of Hydejack's footer), so it appears at the bottom of every layout — including 404, license pages, and any future post. The loader is deferred via requestIdleCallback after window.load so a slow / blocked clustrmaps CDN can never block rendering or Hydejack's drawer init. The idempotency guard (getElementById('clustrmaps') return) keeps push-state navigation from double-injecting the script. Hide-on-fail removes the empty 300x200 box for visitors who block the tracker (uBlock, Pi-hole, NextDNS, …).
